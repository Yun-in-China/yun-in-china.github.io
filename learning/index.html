📖 使用说明 How to use
  1. 先点上面的「我学英语 I'm learning English」按钮 — 这样英语会放在最前面，发音练习也会练英语。
  2. 📇 卡片 — 看单词、拼音和例句。点 🔊 听发音。上面的"速度 Speed"滑块可以把声音调慢。
  3. 🎤 跟读 — 点麦克风按钮，大声读出英语单词，电脑会告诉你发音对不对。✅ 表示说对了！（第一次使用时请允许浏览器使用麦克风）
  4. 🎯 测验 — 看中文，从四个选项里选出正确的英文。
  5. 👂 听力 — 只听声音，选出你听到的英语单词。锻炼耳朵！
  6. 📅 今日十词 — 每天学十个词：先学习，再测验。每天坚持，保持连续记录 🔥！
  7. 卡片上的星星 ★ 表示你已经掌握的词 — 答错会重新练习，答对会加星。
  8. 进度自动保存在浏览器里 — 请一直用同一个浏览器（比如 Safari）。
🎤 The "Say it" button uses your microphone to check pronunciation — works in Safari, Chrome, and Edge (allow mic access when asked). 麦克风按钮：请允许浏览器使用麦克风。
If a voice is missing — Mac: System Settings → Accessibility → Spoken Content → System Voice → Manage Voices（系统设置 → 辅助功能 → 朗读内容）. Windows: Settings → Time & Language → Speech → Add voices.
